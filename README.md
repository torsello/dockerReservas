# Sistema de Reservas

Sistema de Reservas is a java application to manage hotel and customer reservations

## Requirements

Docker: https://www.docker.com/get-started

## Installation

- Open terminal
- Go to the directory where docker-compose is located
- run the next line

```bash
docker-compose up
```
